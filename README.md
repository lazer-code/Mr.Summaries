# Mr.Summaries
## Introduction
Mr.Summaries is a platform built for students.
The platform contains the following:
* Notebooks platform
* Summaries platform
* Calendar & Tasks

The goal of the platform Mr.Summaries is to ease the access for academic knowledge and sharing it with all students on the platform for easy and seamless learning.

The platform comes in 2 variations:
1. Website
2. Application

The application can do everything the website can and extend functionality. 

In the platform, all must be rectangular with rounded corners.
## Goals
* Modern design.
* Memory efficiency
* Fast loading
* Multidevice support

## The Notebook Platform
### Introduction
The notebook platform is a platform for digital notes taking in a notebook form.

### Home screen
When opening the notebook platform, the home screen must get shown first.
The home screen shows the filesystem and a list of the 10 last opened notebooks with the option of creating a new notebook and folder.

### Filesystem
The filesystem can support nested folders and labels.

### Design
Each notebook is in the form of A4 pages.
A page can be for handwrite and text.

### Features
Each notebook can have headers, titles, links, math equations (Latex), pictures, videos,...
Each notebook start with a starter page, then a context menu according to the headers. (Therefore, is there is no headers, there will not be a page for the context menu)

### Writing
Writing in the notebook must support stylus features.
There must be a window for stylus settings in which the user will be able to adjust the functionality of the buttons on their stylus.
By default, the button on the stylus will trigger a stroke eraser as long as it is being pressed.

There must be the following functionality for writing:
1. Pen
2. Eraser
3. Highlighter
4. Shape (Auto shape)

The notebook must be able to redo and undo changes.

## The summary platform
### Introduction
The summary platform is a platform in which all the summaries students upload from all the classes are being showed.

### Home Screen
When opening the summary platform, the home screen must get shown first.
The home screen contains:
* A search bar for course number or name or a subject.
* Lists of the latest uploaded summaries.
* A button for summary upload.

### Summary Block
A summary block is a block which shows the name of the course, upload time, title of summary, the author and the beginning of the summary.

### Summary page
When selecting a summary to view on the summary platform home screen, it should open the summary in a rounded corners rectangular pop-up page which shows the name of the summary, the course, the author, upload time and the full summary.
There must be a close button which closes the pop-up window and returns to the home screen.

Each summary can contain headers, titles, pictures, tables, math equations, handwriting and more...

## Calendar and Tasks
A view for a calendar like google's and tasks presented in the calendar.
